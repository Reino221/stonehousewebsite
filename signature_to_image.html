Step 1: Create Screenshot

First, we need to capture your email signature as an image. Open your current email_signature.html file and take a screenshot.

Step 2: Use this template with your image

Once you have the screenshot saved as "email_signature.png", use this HTML code in Outlook:

<div style="position: relative; display: inline-block;">
    <!-- Main signature image -->
    <img src="email_signature.png" alt="Reino Fourie - Stonehouse Holdings" 
         style="display: block; border: none; width: 600px; height: auto;">
    
    <!-- Clickable website area -->
    <a href="https://www.stonehousegroup.co.za" target="_blank" 
       style="position: absolute; left: 250px; top: 90px; width: 250px; height: 20px; 
              text-decoration: none; display: block;"></a>
    
    <!-- Clickable LinkedIn icon -->
    <a href="https://www.linkedin.com/in/reino-fourie-2059b7307?utm_source=share&utm_campaign=share_via&utm_content=profile&utm_medium=ios_app" 
       target="_blank" 
       style="position: absolute; left: 250px; top: 135px; width: 35px; height: 35px; 
              text-decoration: none; display: block; border-radius: 50%;"></a>
    
    <!-- Clickable Facebook icon -->
    <a href="https://www.facebook.com/share/19eD2Zxufi/?mibextid=wwXIfr" 
       target="_blank" 
       style="position: absolute; left: 300px; top: 135px; width: 35px; height: 35px; 
              text-decoration: none; display: block; border-radius: 50%;"></a>
    
    <!-- Clickable WhatsApp icon -->
    <a href="http://wa.me/27645598007" 
       target="_blank" 
       style="position: absolute; left: 350px; top: 135px; width: 35px; height: 35px; 
              text-decoration: none; display: block; border-radius: 50%;"></a>
</div>

Step 3: Instructions

  1. Create the image:
    • Open email_signature.html in your browser
    • Take a full screenshot of the signature
    • Save it as email_signature.png (exactly this name)
    • Make sure the image is about 600px wide
  2. Set up in Outlook:
    • Go to Outlook Settings > Mail > Compose and reply
    • Create new signature
    • Copy the HTML code above
    • Paste it in the signature editor
    • Upload your email_signature.png image
  3. Adjust coordinates if needed:
    • The left and top values might need adjustment
    • Test clicking on different areas to ensure links work
    • Modify the pixel values if clickable areas don't align perfectly
